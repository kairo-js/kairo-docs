# Kairo JavaScript API Reference

JavaScript API packages for building add-ons in Minecraft Bedrock Edition.

## Packages

### [@kairo-js/router](/api/router)

`0.3.0-beta.0`

Inter-addon communication framework. Enables API calls, hooks, and custom events across multiple behavior packs using `router` and `KairoApiRegistry`.

### [@kairo-js/properties](/api/properties/)

`1.2.1`

Type definitions package: addon properties, versioning, Minecraft dependencies, supported tags, and more.

### [@kairo-js/utils](/api/utils/)

`1.2.1`

Utility functions: semantic version comparison, JSON parsing, timestamp validation, and seeded random number generation.

# MinecraftModule

`import { MinecraftModule } from '@kairo-js/properties'`

Minecraft 모듈 이름 상수를 정의하는 열거형입니다. `MinecraftDependency.module_name`에 사용됩니다.

```typescript
enum MinecraftModule {
  Server = '@minecraft/server',
  ServerAdmin = '@minecraft/server-admin',
  ServerEditor = '@minecraft/server-editor',
  ServerGametest = '@minecraft/server-gametest',
  ServerNet = '@minecraft/server-net',
  ServerUI = '@minecraft/server-ui',
  Debug = '@minecraft/debug-utilities',
  Math = '@minecraft/math',
  Vanilla = '@minecraft/vanilla-data',
}
```

## 멤버

### Server

`Server = '@minecraft/server'`

Minecraft 서버 API 모듈.

---

### ServerAdmin

`ServerAdmin = '@minecraft/server-admin'`

Minecraft 서버 관리 API 모듈.

---

### ServerEditor

`ServerEditor = '@minecraft/server-editor'`

Minecraft 에디터 API 모듈.

---

### ServerGametest

`ServerGametest = '@minecraft/server-gametest'`

Minecraft 게임테스트 API 모듈.

---

### ServerNet

`ServerNet = '@minecraft/server-net'`

Minecraft 서버 네트워크 API 모듈.

---

### ServerUI

`ServerUI = '@minecraft/server-ui'`

Minecraft 서버 UI API 모듈.

---

### Debug

`Debug = '@minecraft/debug-utilities'`

Minecraft 디버그 유틸리티 모듈.

---

### Math

`Math = '@minecraft/math'`

Minecraft 수학 유틸리티 모듈.

---

### Vanilla

`Vanilla = '@minecraft/vanilla-data'`

Minecraft 바닐라 데이터 모듈.

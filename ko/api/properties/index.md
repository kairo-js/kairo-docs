# @kairo-js/properties

애드온 속성 정의 및 메타데이터를 위한 타입 패키지. `router.init()`에 전달되는 `AddonProperties`를 비롯해 시맨틱 버전, 엔진 버전, Minecraft 모듈 의존성 등 애드온 선언에 필요한 모든 타입을 포함합니다.

## 사용 가능한 버전

- `1.2.1`

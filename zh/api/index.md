# Kairo JavaScript API 参考

用于 Minecraft Bedrock Edition 附加包开发的 JavaScript API 包。

## 包

### [@kairo-js/router](/zh/api/router)

`0.3.0-beta.0`

附加包间通信框架。使用 `router` 和 `KairoApiRegistry` 实现多个行为包之间的 API 调用、钩子和自定义事件。

### [@kairo-js/properties](/zh/api/properties/)

`1.2.1`

类型定义包。提供附加包属性、版本管理、Minecraft 依赖项、支持标签等类型定义。

### [@kairo-js/utils](/zh/api/utils/)

`1.2.1`

工具函数包。提供语义化版本比较、JSON 解析、时间戳验证和种子随机数生成等功能。
